MODIFY keyword to change the definition of an existing column in a table

Syntax:

    ALTER TABLE table_name MODIFY COLUMN column_name column_definition;

    mysql> ALTER TABLE users MODIFY COLUMN password VARCHAR(50);
    Query OK, 6 rows affected (0.13 sec)
    Records: 6  Duplicates: 0  Warnings: 0

RUNCATE statement in SQL is used to delete all rows from a table,
but it does not remove the table structure itself (the table still exists, but with no data).

syntax:

    TRUNCATE TABLE table_name;

    mysql> truncate table records;
    Query OK, 0 rows affected (0.05 sec)

    mysql> select * from records;
    Empty set (0.00 sec)

DROP statement is used to completely remove an object from the database

Syntax

    DROP TABLE table_name;

    mysql> drop table records;
    Query OK, 0 rows affected (0.03 sec)

    mysql> show tables;
    +-----------------+
    | Tables_in_users |
    +-----------------+
    | users           |
    +-----------------+
    1 row in set (0.00 sec)
